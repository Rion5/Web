jQuery Basics

Objectives

  • Learn to select elements with $()
  • User .css() to style elements

Selecting with jQuery

First we must select the element/s

$("selectorGoesHere")
  • To select all img tags
    • $("img")
  • To select all elements with class 'sale'
    • $(".sale")
  • To select element with id "bonus"
    • $("#bonus")
  • To select all 'a' (anchor) tags inside of 'li's
    • $("li a")

Manipulating Style

Then we manipulate

the .css() method is jQuery's interface to styling.

$(selector).css(property, value)

Example - This will select the h1 tag, and change the css color to yellow

$("h1").css("color", "yellow");

Select element with id "special" and give it a red border

$("#special").css("border", "2px solid red");

We can also pass in an object with styles var styles = { backgroundColor : "pink:, fontWeight: "bold" }; $("#special").css(styles);
